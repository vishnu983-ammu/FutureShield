Commission Settings

            

              Set advisor and manager commission percentages per product category.
              These are applied automatically when the admin approves a closed sale.
            

          

          
        


        
        

          

            Advisor %
            % of premium paid to the advisor
          

          

            Manager Override %
            % of advisor's commission paid to manager
          

        


        
        

          Product Category
          Advisor %
          Manager Override %
          Preview
        


        
        

          Vehicle products (Two-wheeler, Four-wheeler) have dedicated commission rows.
          They appear automatically once added to Product Settings.
        


        
        

          

            Loading product categories…
          

        


        
        

          
How commission is calculated:

          

            
1. Advisor Commission = Premium Amount × (Advisor % ÷ 100)

            
2. Manager Override = Advisor Commission × (Manager Override % ÷ 100)

            
Example: Premium ₹50,000 · Advisor 5% · Manager Override 20%

            
→ Advisor earns ₹2,500 · Manager earns ₹500 (20% of ₹2,500)
